# 📷 AI 摄影大师

**让每个人都能拍出专业级照片和视频的智能相机**

> 集成摄影基础理论 + 计算机视觉 AI，做你身边的摄影大师 👇

## ✨ 核心功能

### 🤖 AI 智能场景分析
基于**摄影分区曝光法（Zone System）** 实时分析画面亮度分布，自动识别：
- 👤 人像 / 🏔️ 风景 / 🌙 夜景 / 🌗 逆光 / 🏠 室内 等 10+ 种场景
- 判断光照条件：强烈日光 / 阴天 / 室内人工光 / 弱光 / 极暗
- 给出曝光质量评估：过曝警告 / 欠曝提示 / 层次丰富

### 📊 专业相机参数显示
- **EV 值**：实时曝光补偿显示，基于场景自动推荐
- **ISO 显示**：当前感光度指示
- **快门指示**：当前快门速度（基于帧率估算）

### 🎯 智能构图辅助线
- **三分法（Rule of Thirds）**：黄金交叉点高亮
- **黄金分割（Golden Ratio）**：自然视觉焦点引导
- **居中构图**：适合对称场景
- **对角线构图**：引导视觉延伸

### 🛠️ 专业辅助工具
| 功能 | 说明 |
|------|------|
| 📈 **实时直方图** | 亮度分布实时可视化，阴影/中间调/高光分区 |
| 📐 **水平仪** | 借助设备陀螺仪检测水平，辅助建筑/风景拍摄 |
| 🦓 **斑马纹（Zebra）** | 专业摄像功能，红色标记过曝区域（亮度>210） |
| 🎯 **对焦峰值（Focus Peaking）** | Sobel 边缘检测，红色高亮合焦区域 |
| 👆 **触摸对焦** | 点击画面任意位置触发对焦反馈环 |

### 📸 五种拍摄模式
| 模式 | 说明 |
|------|------|
| **🤖 AI 智能** | 自动识别场景 + 自动调整参数 |
| **👤 人像** | 三分法构图，暖调柔光建议 |
| **🏔️ 风景** | 黄金分割构图，大景深建议 |

### 🌅 场景参数知识库
基于摄影经典参数理论，每种场景对应最佳参数：

| 场景 | EV | ISO | 白平衡 | 建议 |
|------|----|-----|--------|------|
| ☀️ 晴天户外 | 0 | 100 | 晴天 | 高饱和、丰富层次 |
| ⛅ 阴天 | -0.5 | 自动 | 阴天 | 低饱和灰调 |
| 🌙 夜景 | -1 | 800 | 自动 | 长曝光三脚架 |
| 🌗 逆光 | +1.5 | 自动 | 自动 | 点测光+补光 |
| 🏠 室内 | 0 | 400 | 暖调 | 靠近窗户利用自然光 |
| ⚡ 运动 | 0 | 800 | 自动 | 高快门凝固瞬间 |
| 🎆 烟花 | -2 | 200 | 白炽灯 | 三脚架长曝光 |

## 🚀 快速开始

```bash
git clone https://github.com/DSC729/ku.git
cd ku
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

> ⚠️ 相机功能需要 HTTPS 或 `localhost` 环境

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| TensorFlow.js + BlazeFace | 人脸检测 |
| TensorFlow.js + MobileNet | 场景分类 |
| Canvas API | 直方图/斑马纹/对焦峰值 |
| MediaStream API | 相机/视频录制 |
| DeviceOrientation API | 水平仪 |
| Web Audio API | 测光（规划中）|
